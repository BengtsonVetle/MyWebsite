Om meg

Jeg har en grunnleggende forståelse for;

- Oppsett, oppdatering og vedlikehold av nettverk

- Ulike operativsystemer som Linux, Windows og MacOS

- VMWare (virtualisering)

- Cisco Packet Tracer

- Microsoft Server Technologies

- Cloud computing (spesielt Microsoft Azure)

- IT Security

- samt diverse software og programmer
