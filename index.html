Om meg

Heisann! Mitt navn er Vetle Bengtson, og jeg studerer Nettverk- og systemadministraasjon ved Noroff Fagskole, hvor jeg er ferdig utdannet til sommeren. Jeg er 25 år gammel, og er oppvokst og bor i Sandnes. Jeg har alltid hatt en interesse for alle mulige tekniske dippedutter, og bestemte meg i 2023 å gjøre hobby om til jobb. Gjennom utdannelsen min har jeg lært masse nytt, og jeg har fortsatt masse å lære. Jeg har en grunnleggende forståelse for; - Datakomponenter - Nettverkskomponenter - Oppsett, oppdatering og vedlikehold av nettverk - Ulike operativsystemer som Linux, Windows og MacOS - VMWare (virtualisering) - Cisco Packet Tracer - Microsoft Server Technologies - Cloud computing (spesielt Microsoft Azure) - IT Security - samt diverse aktuelle software og programmer
